# Ambulatorio Bocceas

Sito web ufficiale dell'Ambulatorio Medico Bocceas.

## Descrizione

Sito responsive per ambulatorio medico con le seguenti sezioni:
- **Home**: Presentazione del servizio con CTA (Chiama/WhatsApp)
- **Servizi**: Lista dei servizi offerti
- **Team**: Presentazione del team medico
- **Contatti**: Informazioni di contatto e mappa Google Maps

## Tecnologie utilizzate

- HTML5
- CSS3 (con animazioni gradient e fade-up)
- JavaScript vanilla (menu responsive)

## Hosting

Hostato su GitHub Pages: https://ambulatorioodontoiatricoboccea.github.io

## Struttura

```
.
├── index.html          # Homepage
├── servizi.html        # Pagina servizi
├── team.html           # Pagina team
├── contatti.html       # Pagina contatti
├── css/
│   └── style.css       # Stili globali
├── js/
│   └── script.js       # JavaScript (menu toggle)
└── images/             # Cartella per le immagini
```

## Note

- Aggiungi le tue immagini nella cartella `/images/`
- Aggiorna i numeri di telefono e email nei file HTML
- Configura correttamente l'embed di Google Maps in `contatti.html`
